# 口算小程序系统用例分析 - 快速查阅指南

## 📌 核心要点速览

### 系统参与者（2 个）

- **学生**：主要使用者，可进行训练、管理错题、查看统计
- **教师**：有限权限，仅可查看数据概览

### 系统用例（23 个）

按模块分类：

- **认证模块**（5 个）：登录、注册、登出、认证
- **首页模块**（1 个）：首页概览
- **训练模块**（8 个）：参数配置、题目生成、答题、判题
- **错题模块**（5 个）：查看、筛选、管理
- **统计模块**（4 个）：多维度统计分析

## 💡 关键业务规则

| 规则名           | 条件                                     | 结果                               |
| ---------------- | ---------------------------------------- | ---------------------------------- |
| **等级晋升**     | 最近 20 题正确率 ≥ 90% AND 当前等级 < 10 | 自动晋升至下一等级                 |
| **答题计分**     | 基础分（0 或 100）+ 耗时奖励（0～20）    | 最终得分 0～120                    |
| **答题错误判定** | 用户答案 ≠ 标准答案                      | 记录错题 + 保存到错题库            |
| **账号锁定**     | 密码连续输入错误 3 次                    | 锁定 30 分钟，无法登录             |
| **会话过期**     | 用户 15 分钟无操作                       | 需要重新登录                       |
| **错题改正**     | 用户点击"标记改正" + 确认                | is_corrected = 1，从未改正列表移除 |

## ⚠️ 常见异常处理

| 场景                  | 处理方式                             |
| --------------------- | ------------------------------------ |
| **网络断开**          | 缓存状态 → 提示用户 → 恢复网络后继续 |
| **Token 过期**        | 自动刷新 → 失败则跳转登录            |
| **题目生成失败**      | 重试 → 提示用户更换参数              |
| **答题保存失败**      | 本地缓存 → 网络恢复后上传            |
| **密码连续错误 3 次** | 锁定账号 30 分钟                     |
| **后端服务异常**      | 显示 "服务异常，请稍后重试"          |
