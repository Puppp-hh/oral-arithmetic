# 口算小程序系统 - 用例关系详细分析

## 1. 用例关系矩阵

### 1.1 包含关系（Include）

## 2. 用例交互序列

### 2.1 登录与首页展示

```
参与者：学生
UC-01（学生登录） → UC-00（执行认证） → UC-04（首页）

时间序列：
1. 学生打开应用
2. 学生进入登录页
3. 学生输入账号密码并点击登录
4. 前端向后端发送登录请求（UC-01）
5. 后端执行身份验证（UC-00）
   5a. 验证账号密码（UC-00A）
   5b. 生成认证令牌（UC-00B）
6. 前端保存 Token
7. 系统请求学生摘要数据
8. 系统展示首页（UC-04）
   8a. 可选：展开查看个人信息（UC-04A）
   8b. 可选：查看等级进度（UC-04B）
```

### 2.2 完整的训练过程

```
参与者：学生
UC-05 → UC-06 (loop) → UC-07 (loop) → UC-08

详细时间序列：
1. 学生点击"开始训练"（UC-05）
2. 系统展示参数配置表单（UC-05A）
   - 选择难度等级（UC-05A1）
   - 选择题型（UC-05A2）
   - 设置题目数量（UC-05A3）
3. 系统向后端请求生成题目（UC-05B）
4. 后端返回题目列表
5. 对每一道题目执行循环：
   i. 系统展示题目（UC-06）
      - 启动计时器（UC-06A）
      - 学生输入答案
   ii. 学生点击提交（UC-07）
       - 系统停止计时
       - 后端评判答案（UC-07A）
       - 系统显示结果（UC-07B）
       - 学生查看结果
   iii. 如果未完成所有题目，返回到第5步（i）
6. 学生答完所有题目后，系统结束训练（UC-08）
   - 计算统计数据
   - 更新学生档案
   - 检查等级晋升
   - 展示本次汇总（UC-08A）
7. 学生点击"返回首页"或"继续训练"
```

### 2.3 错题管理流程

```
参与者：学生
UC-09 → UC-09A → {UC-09B, UC-09C} → {UC-09D, UC-09E}

详细时间序列：
1. 学生点击"查看错题本"（UC-09）
2. 系统请求错题列表
3. 系统展示错题本列表（UC-09A）
4. 学生可选择：
   - 执行筛选（UC-09B）：按状态、题型、难度、时间筛选
5. 学生点击某道题目
6. 系统展开错题详情（UC-09C）
   - 展示题干、标准答案、个人答案
   - 展示错题统计（共做错几次）
7. 学生可选择执行以下操作之一：
   - 标记错题已改正（UC-09D）
   - 删除错题记录（UC-09E）
   - 或返回列表继续查看其他错题
```

## 5. 异常与边界情况处理

### 5.1 网络异常处理

| 用例               | 异常情况         | 处理策略                         |
| ------------------ | ---------------- | -------------------------------- |
| 所有涉及网络的用例 | 网络请求失败     | 自动重试 3 次，提示用户后仍失败  |
| UC-05～UC-08       | 训练中网络断开   | 缓存当前答题状态，恢复网络后继续 |
| UC-09              | 加载错题列表失败 | 显示上次缓存数据，提示更新异常   |
| UC-11              | 加载统计数据失败 | 骨架屏提示，支持下拉刷新         |

### 5.2 认证异常处理

| 异常情况          | 处理策略                         |
| ----------------- | -------------------------------- |
| Token 过期        | 自动刷新 Token，失败则跳转登录页 |
| Token 无效        | 清除本地登录状态，跳转登录页     |
| 密码连续错误 3 次 | 锁定账号 30 分钟                 |
| 账号不存在        | 提示用户账号错误或建议注册       |

### 5.3 业务异常处理

| 异常情况         | 处理策略                   |
| ---------------- | -------------------------- |
| 题目生成失败     | 更换参数重试，提示用户     |
| 答题记录保存失败 | 缓存到本地，网络恢复后上传 |
| 等级数据不一致   | 从服务器重新获取权威数据   |
| 错题删除失败     | 提示用户稍后重试           |
